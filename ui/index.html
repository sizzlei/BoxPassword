2단계 인증 (TOTP) 설정

사이트에서 받은 otpauth:// URL 또는 base32 시크릿을 입력하세요. 알고리즘 SHA-1 / 30초 / 6자리만 지원합니다.

그룹 관리

항목을 카테고리로 묶을 수 있습니다. 그룹을 삭제해도 항목은 미분류로 이동하며 사라지지 않습니다.

설정

마우스·키 입력이 없으면 지정 시간 뒤에 Vault가 자동 잠금됩니다.
macOS → Keychain (Touch ID 가능) · Windows → Credential Manager · Linux → Secret Service. 저장소는 OS별로 따로 관리되며, 끄면 즉시 항목이 삭제됩니다. 보안 모델은 OS 로그인 인증에 위임됩니다.
복사 직후 BoxPassword 메모리엔 SHA-256 해시만 남고 평문은 즉시 폐기됩니다. 타이머 만료 시 클립보드 내용이 그대로면 비웁니다(사용자가 다른 것을 복사했다면 건드리지 않습니다).

CSV 가져오기

1Password / Bitwarden / Chrome 등에서 export한 CSV 내용을 그대로 붙여 넣으세요. 헤더에서 name/title, username, password, url, notes, folder, totp 등을 자동 인식합니다. folder 열의 값은 그룹으로 자동 생성됩니다.

백업 내보내기

마스터 비밀번호로 봉인된 .bpvault 파일을 만듭니다. 비밀번호 외에는 어떤 정보도 평문으로 노출되지 않습니다.
