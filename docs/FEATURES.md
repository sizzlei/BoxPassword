# BoxPassword 기능 안내

일상 사용 시 알아두면 좋은 기능을 정리했습니다. 각 항목은 어디서 어떻게 쓰는지에 초점을 둡니다.

## 첫 실행 — Vault 만들기

앱을 처음 켜면 마스터 비밀번호 설정 화면이 뜹니다. 8자 이상으로 두 번 같게 입력 → "Vault 만들기".
이후 모든 항목은 이 비밀번호로 봉인됩니다.

> ⚠️ 분실 시 복구 경로가 없습니다. 안전한 비밀번호 매니저로 옮겨 적거나 인쇄해 보관하세요.

## 항목 추가

상단 가운데 리스트 상단의 **+ 버튼** 또는 ⌘N. 모달이 뜨면 제목·사용자명·URL·메모·패스워드를 입력합니다.

- **🎲 패스워드 생성기**: 모달 안에서 펼쳐 보세요. Random/Passphrase 탭, 길이/문자 클래스/가독성/연속 제한 옵션. 후보 5개를 생성하고 "사용" 클릭으로 적용.
- **그룹**: 드롭다운에서 미리 만든 그룹에 분류 가능.
- **메모**: 자유 텍스트. 봉인 저장.

## 항목 사용 — 복사 / 보기 / 자동 입력

상세 패널의 패스워드 행에서:

- 값 영역 클릭 또는 ⚡ 자동 입력 → 3초 카운트다운 후 OS 키스트로크로 입력 필드에 직접 타이핑.
- 👁 보기/숨기기 → 마스킹 토글.
- 📋 복사 → OS 클립보드(30초 자동 클리어, 기본).

리스트 행에 마우스를 올리면 ⚡/📋 단축 아이콘이 뜨고, 선택 변경 없이 즉시 동작합니다.

### 자동 입력 모드 (설정 → 자동 입력 모드)

- **패스워드만 입력** (기본)
- **사용자명 → Tab → 패스워드**
- **사용자명 → Tab → 패스워드 → Enter**

ID/PW 폼이 같은 화면에 있는 사이트에서 매우 유용합니다.

> macOS는 첫 자동 입력 시 "손쉬운 사용" 권한을 요구합니다. 시스템 설정 → 개인정보 보호 및 보안 → 손쉬운 사용에서 앱(또는 dev 모드의 Terminal/cargo)을 허용해 주세요.

## 메뉴바 트레이

상단 메뉴바에 작은 BoxPassword 아이콘이 상주합니다.

- **잠금/해제 상태**가 아이콘 배지(빨강 X / 초록 ✓)로 표시.
- **좌클릭** → 메인 창 보이기.
- **우클릭(또는 control-click)** → 컨텍스트 메뉴:
  - BoxPassword 열기 / 빠른 검색 (⌘⇧K)
  - ★ 즐겨찾기 (각 항목 → 복사/자동 입력 서브메뉴)
  - 🕒 최근 (마찬가지)
  - 🎲 비밀번호 즉시 생성 (랜덤 20/32자, 패스프레이즈 5/7단어)
  - Vault 잠그기 / 클립보드 비우기 / 종료

메인 창 빨강 닫기 버튼은 종료가 아니라 **숨기기**입니다. 트레이에서 다시 띄울 수 있습니다.

## Quick Search 팝오버 (⌘⇧K)

어떤 앱에 있든 OS 전역 단축키 **⌘⇧K** 로 검색 팝오버를 띄울 수 있습니다.

- 타이핑 → 매칭 항목 즉시 필터링.
- ↑↓ 이동 → Enter 복사 / ⌘Enter 자동 입력 → 팝오버 자동 닫힘.
- Esc 닫기.

Vault가 잠겨 있으면 "메인 창에서 잠금 해제하세요" 안내가 표시됩니다.

## TOTP 2단계 인증

상세 패널 "2단계 인증 (TOTP)" 카드에서 시드 설정 가능.

- "TOTP 추가" → 사이트가 보여준 `otpauth://...` URL을 그대로 붙여 넣거나 base32 시크릿(예: `JBSWY3DPEHPK3PXP`) 입력.
- 저장 시 시드는 vault에 봉인된 상태로 보관됩니다.
- 6자리 코드가 1초마다 갱신되고, 도넛이 남은 시간을 시각화. 클릭하면 클립보드 복사.

> 알고리즘 SHA-1 / 30초 / 6자리만 지원합니다(가장 일반적). 다른 설정의 사이트는 v0.2에서 다룹니다.

## 버전 이력 / 롤백

비밀번호를 바꿀 때마다 새 버전이 append-only로 쌓입니다. 상세 하단 "버전 이력" 카드에서:

- 각 버전마다 ⚡ 자동 입력 / 👁 보기 / 📋 복사.
- 비현재 버전 우측의 ↶ 클릭 → 그 시점 비밀번호로 새 버전 생성(이전 버전들은 그대로 보존).

## 즐겨찾기 / 그룹 / 정렬

- 상세 헤더 ★ 아이콘 클릭으로 즐겨찾기 토글.
- 사이드바 "그룹" → "관리" 버튼으로 그룹 생성·이름 변경·색상 순환·삭제. 새 항목 추가 시 드롭다운으로 분류.
- 가운데 리스트 상단 정렬 드롭다운: 제목순 / 최근 수정 / 즐겨찾기 우선.

## 건강 검진

사이드바 "건강 검진"을 클릭하면 우측 패널이 대시보드로 전환됩니다.

- 평균 강도 (zxcvbn 0~4점 평균)
- 약한 비밀번호 (점수 ≤ 1)
- 중복 사용 그룹 (SHA-256 해시로 분류, 해시 자체는 외부 노출 X)
- 1년 이상 오래된 비밀번호

각 항목을 클릭하면 일반 보기로 이동해 바로 "변경" 흐름으로 이어집니다.

## 백업 / 복구

사이드바 → **백업 내보내기** → 마스터 비밀번호 확인 → 저장 위치 다이얼로그 → `.bpvault` 파일.
이 파일은 마스터 비밀번호 없이 열 수 없습니다.

사이드바 → **백업 복구** → 파일 선택 → 비밀번호 입력 → 기존 vault는 자동으로 `vault.db.bak`로 보존되고, 새 vault로 교체. 잠금 화면이 뜨면 마스터 비번으로 해제.

## 자동 잠금 / 클립보드 / 테마

설정(사이드바)에서 모두 조정 가능:

- 자동 잠금: 1분 / 5분(기본) / 15분 / 30분 / 1시간 / 사용 안 함. 마우스·키보드 활동마다 리셋.
- 창 포커스를 잃을 때 즉시 잠금 토글.
- 클립보드 자동 비우기: 5/10/20/30/60/120초.
- 자동 입력 모드(앞서 설명).
- 라이트/다크 테마는 상단바 ☼/☾ 버튼 — auto(OS 추종)/light/dark 순환.

## 마스터 비밀번호 변경

설정 모달 하단 **"마스터 비밀번호 변경…"** 버튼.
현재 비밀번호와 새 비밀번호 + 확인 입력 → 변경. 즉시 옛 비번은 무효화되고, 항목 데이터는 그대로 유지됩니다.

## 보관 위치

- macOS: `~/Library/Application Support/BoxPassword/vault.db` (기본)
- 환경 변수 `BOX_PASSWORD_VAULT_PATH` 로 다른 경로 사용 가능.
- 이 파일을 iCloud Drive 폴더에 두면 자연스러운 동기화 효과(다만 충돌 머지는 비범위).
