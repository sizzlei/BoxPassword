# BoxPassword 통합 테스트 체크리스트

릴리스 빌드 또는 큰 변경 직후 한 번 훑으면 좋은 항목들입니다.
각 항목은 5초~1분 이내로 확인 가능합니다.

## 사전 준비

```bash
cargo test --workspace
cargo clippy --workspace --all-targets -- -D warnings
cargo fmt --check
cd crates/bp-app && cargo tauri dev
```

## 1. 첫 실행 / 잠금 / 해제

- [ ] 새 vault 경로(`BOX_PASSWORD_VAULT_PATH=/tmp/test.db`)로 띄우면 셋업 화면이 뜨고 빨간 vault 로고가 보인다.
- [ ] 8자 이상 같은 비번 2회 입력 → 즉시 잠금 해제 상태로 진입.
- [ ] ⌘L 또는 트레이 "Vault 잠그기" → 잠금 화면으로 즉시 전환.
- [ ] 사이드바 하단 카드의 dot이 잠금 시 빨강 / 해제 시 초록으로 바뀐다.

## 2. 항목 CRUD

- [ ] **+ 버튼** 또는 ⌘N → 새 항목 모달.
- [ ] 제목·사용자명·URL·메모·변경주기 입력 후 저장.
- [ ] 리스트에 즉시 등장, 사이드바 하단 카운트 증가.
- [ ] 항목 선택 → 상세 패널에 노트가 보이고, "저장됨 ✓" 토스트 후 갱신 확인.
- [ ] 상세 헤더의 ★ 클릭 → 즐겨찾기 토글, 사이드바 즐겨찾기 카운트 +1.
- [ ] 🗑 삭제 → 자체 확인 다이얼로그 → 삭제 후 다음 항목 자동 선택.

## 3. 패스워드 동작 (자동입력 없음, 복사 전용)

- [ ] 패스워드 값 영역 클릭 → 30초 자동 클리어 토스트.
- [ ] 👁 보기 토글 → 마스킹 해제/복원.
- [ ] 📋 복사 아이콘 → 동일.
- [ ] 상단바 클립보드 pill이 황금색 도넛으로 30초 카운트다운.
- [ ] "비우기" 클릭 → 즉시 클리어, pill 사라짐.
- [ ] 리스트 행에 마우스 올리면 📋 복사 아이콘만 나타난다(⚡ 자동입력 제거됨).

## 4. 변경 이력

- [ ] 상세에서 "변경" 버튼 → 새 비번 입력 → 저장.
- [ ] 버전 이력에 v1(이전) + v2(현재) 표시. 현재에 "현재" 배지.
- [ ] v1 행의 ↶ 복원 → 자체 확인 다이얼로그 → v3 생성, 현재값이 v1과 동일.
- [ ] 각 버전 행에는 👁 보기 / 📋 복사 / ↶ 복원만 노출(⚡ 제거됨).

## 5. 그룹 / 정렬 / 검색

- [ ] 사이드바 그룹 "관리" → 이름 + 색 선택 → 생성. 사이드바에 노출.
- [ ] 새 항목 모달의 그룹 드롭다운에 추가됨.
- [ ] 상세 헤더 그룹 pill → 다른 그룹으로 변경 → 즉시 반영.
- [ ] 그룹 관리에서 인라인 이름 변경 / 색상 순환 / 삭제(미분류로 이동).
- [ ] 가운데 검색 박스 ⌘F → 타이핑 시 디바운스 필터링.
- [ ] 정렬 드롭다운: 제목순 / 최근 수정 / 즐겨찾기 우선 정확히 적용.

## 6. 패스워드 생성기 / TOTP

- [ ] 새 항목 모달의 🎲 → Random 탭 → 길이 슬라이더 변경 → "5개 후보 생성" → 강도 바와 함께 표시.
- [ ] 후보 "사용" 클릭 → 비밀번호 칸 채워짐 + 라이브 강도 바.
- [ ] Passphrase 탭 → 단어 수 5, 구분자 "-", 대문자 → 생성.
- [ ] 트레이 🎲 즉시 생성 4종 → 클릭 시 즉시 클립보드 + 알림.
- [ ] 상세 TOTP 카드 "TOTP 추가" → otpauth URL 붙여넣기 → 저장 → 6자리 코드와 황금 도넛 표시.
- [ ] SHA-256 / 60초 / 8자리 인 otpauth URL도 받아 정상 코드 생성(자릿수가 8로 표시).
- [ ] 코드 클릭 → 클립보드 복사. 1초마다 갱신.
- [ ] TOTP 제거 → 카드가 "TOTP 추가" 버튼만 남는 상태로.

## 7. 백업 / 복구 / CSV 가져오기

- [ ] 사이드바 "백업 내보내기" → 비번 재확인 → 저장 다이얼로그 → `.bpvault` 생성.
- [ ] 항목 한두 개 삭제.
- [ ] "백업 복구" → 그 파일 → 같은 비번 → 토스트 "이전 파일: …/vault.db.bak". 잠금 화면 진입.
- [ ] 잠금 해제 → 삭제했던 항목 복원 확인.
- [ ] 사이드바 "CSV 가져오기" → 1Password 또는 Bitwarden export CSV 붙여넣기 → 가져오기.
- [ ] 토스트로 추가/건너뜀/새 그룹 카운트 표시. 항목 리스트에 즉시 등장.
- [ ] folder 열의 값이 자동으로 그룹으로 생성됨.

## 8. 트레이 / Quick Search / 글로벌 단축키

- [ ] macOS 메뉴바에 빨간 vault 아이콘 등장. 잠금 시 빨간 자물쇠 배지 / 해제 시 초록 ✓.
- [ ] 좌클릭 → 메인 창 보이기. 우클릭 → 메뉴.
- [ ] 즐겨찾기/최근 서브메뉴에서 항목 클릭 → 자동 복사 + 토스트.
- [ ] 🎲 비밀번호 즉시 생성 → 클립보드 + 알림.
- [ ] ⌘⇧K → Quick Search 팝오버 등장. 타이핑 → 필터 → Enter 복사 → 자동 닫힘.
- [ ] 메인 창 빨강 닫기 버튼 → 종료 대신 hide. 트레이로 다시 열 수 있음.

## 9. 자동 잠금 / 알림 / 시스템 슬립

- [ ] 설정 → 자동 잠금 "1분" → 1분 비활동 → 잠금 화면 + 토스트 + (켜져있으면)알림.
- [ ] 설정 → 창 포커스 잃을 때 잠금 토글 → 다른 앱으로 ⌘Tab → 즉시 잠금.
- [ ] 설정 → 알림 ON → macOS 권한 요청 → 허용 후 자동 잠금 알림 도착.
- [ ] 노트북 닫기 → 잠시 후 열기 → 자동 잠금 + "시스템 복귀 감지" 토스트.
- [ ] 상단바 자동 잠금 카운트다운 30초 이하면 노란/빨간 dot로 단계 경고.

## 10. 변경 주기 알람

- [ ] 새 항목 추가 시 변경 주기 30일 선택.
- [ ] `BOX_PASSWORD_VAULT_PATH` 로 임시 vault에 만든 항목의 `entries.updated_at` SQL 로 과거(31일 전)로 조정.
- [ ] 잠금 → 해제 시 토스트/알림으로 "1개가 변경 기한을 지났습니다" 표시.
- [ ] 상세 헤더 sub-title 옆에 빨간 "⚠ N일 지남" pill.
- [ ] pill 클릭 → 변경 주기 즉시 수정 다이얼로그.
- [ ] 건강 검진의 "오래된 비밀번호 (1년+)" 섹션이 별개로 동작(rotation은 항목별, aged는 1년 절대 기준).

## 11. 마스터 비밀번호 변경

- [ ] 설정 → "마스터 비밀번호 변경…" → 옛/새/확인.
- [ ] 같은 비번 입력 / 8자 미만 / 확인 불일치 → 각각 명확한 메시지.
- [ ] 변경 후 잠그고 새 비번으로 해제 → 성공. 옛 비번 → InvalidMaster.

## 12. 테마 / 빈 상태 / 일러스트

- [ ] 상단바 ☼/☾ → auto → light → dark 순환. 모달/토스트/리스트 모두 따라옴.
- [ ] OS 시스템 모드 변경 → auto 상태면 즉시 추종.
- [ ] 항목 0개 또는 검색 결과 없음 → 친절한 메시지.
- [ ] 항목 미선택 상태 → 큰 자물쇠 일러스트 + 키보드 힌트(⌘N / ⌘F / ⌘L).

## 알려진 제한 (의도된 범위)

- 자동 입력(키스트로크) 기능은 제거됨. 모든 패스워드 전달은 클립보드 경유.
- 코드사인 미적용 — 다른 Mac에 옮기면 `xattr -dr com.apple.quarantine` 한 번 필요.
- 시스템 슬립 감지는 NSWorkspace API 가 아닌 Instant 갭 휴리스틱(15초+).
- TOTP는 SHA-1/256/512, period 1-3600초, 자릿수 6-10. HOTP / 카운터 기반은 미지원.
- 멀티 vault / 첨부 파일 / 브라우저 확장은 v0.2 후보.
